Utopia
Typeface, Website
‘12

A typeface designed using only static HTML divs and CSS positioning. Utopia translates the ideas of the De Stijl movement to the internet, asking the question if the internet is a utopia or not. The typeface recreated De Stijl paintings, and was categorized based on Alexander Galloway's internet structure.
